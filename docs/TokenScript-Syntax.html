TokenScript Syntax

The most important declarative terms used in the TokenScript XML file

Note:

A TokenScript file consists of Token Cards, presently in JavaScript and HTML, overlayed with XML mark-up.

More on that see TokenScript File.

The root element of a TokenScript File can be one of the following:

  • <cards> element
  • <card> element
  • <tokens> element
  • <token> element
Tip:

If a Token Issuer issues multiple tokens, it's important to define them in a single TokenScript file using <tokens> element, for TokenScript release management reason.

A list of TokenScript elements can be found - as a work in progress - in TokenScript XML Markup elements
