Deploy TokenScript

After you created a TokenScript, you want wallets to use it. Learn how wallets can get a TokenScript.

To be operated in a wallet or any other user agent software, the TokenScript engine is required. Currently it is only integrated in Alphawallet and forks of Alphawallet. In the future a library and a web library will be available for easy integration.

If your TokenScript is signed and uploaded to the TokenScript repository, itâ€™s easy for Alphawallet to know about it: If the user receives a token, Alphawallet will scan the repository for a TokenScript and install it.
