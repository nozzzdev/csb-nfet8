    Filter

    In TokenScript, filter is like the search query of tokens

    A filter is used for a TokenScript engine to find tokens, Data Objects (such as Ethereum Events) and attestations by its attributes.

    It's format resembles a SQL search query.

    Find all tokens located in Sydney

    Suppose the token we are working with is AirBnB Booking token, where each token represents an AirBnB booking. Let's say AirBnB token has an attribute locality which is a string. The following filter would result in all of the bookings made in Sydney:
    locality=Sydney
