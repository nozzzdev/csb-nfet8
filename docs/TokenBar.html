    TokenBar

    TokenBar shows what tokens can be used in the current web session.

    A Token Bar is a UI element that shows users' tokens that can be used on a website.

    Restriction: TokenBar is an in-development feature. There is no released AlphaWallet or TokenScript engine that supports it yet.

    In a Browser with TokenScript support, a token bar is accessed in a samiliar fashion as the Bookmarks, which is a list of tokens in their ItemView. Clicking the token would "activate it", meaning making the website aware that the user has that token.

    With TokenScript Bootstrap Library the Token Bar is rendered on the upper-right cornor of the screen similar to "Login with Google".
